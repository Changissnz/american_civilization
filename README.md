Wankelmütige, kleinliche, hinterhältige Amerikaner.  

Die US-Führung will, dass die Mittelschicht verarmt. Die Unterschicht wünscht sich, dass die Mittelschicht den Verstand verliert. Die Mittelschicht wiederum wünscht sich, dass die Oberschicht verschwindet – ohne Geld, ohne Macht und am besten ganz von dieser Welt.  

Warum müssen die meisten Menschen beschämt werden, damit sich Homosexuelle wohlfühlen und Frauen Führungspositionen übernehmen können? Wegen der Geschichte. Einer schlechten Geschichte, einer schlechten Führung. Eine Führung, die so mittellos wäre wie ein Obdachloser, wenn sie sich nicht die Dinge anderer Menschen aneignen könnte. Es ist traurig. Traurig, dass, damit eine historisch marginalisierte Gruppe Gehör findet und mehr Rechte erhält, die Angst von ihr auf andere Menschen gelenkt werden muss.
